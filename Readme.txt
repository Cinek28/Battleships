Project for java programming course.
Basic 2 player battleships game with client-server
connection and interface using JavaFX.

How to run:

In folder Battleships (in terminal) type:

javac Battleships.java
java Battleships

WARNING:
Project is not finished! There is a problem with NullPtr exception.

In game: 
Server: Type port number you want to create server on and click start server.
Wait for other player to connect.

Client: Type port number and IP address of created server and connect.

Click mouse on the area you want to shoot (left board is for the player, right for opponent).
